Detailed Description

The SkyeTeachAndRepeat class is the Teach and Repeat Library, it provides functionalities to change mode, assign new actions to be repeated, initialize and update the teach and repeat object parameters. Then, during the execution, only one function should be called i.e. ExecuteTeachAndRepeat.
