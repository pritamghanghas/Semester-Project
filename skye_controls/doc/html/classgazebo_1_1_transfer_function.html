This class can be used to apply a filter on a signal. Short reveiw : continous time system: dx(t) = A*x(t) + B*u(t) discretized system (ZoH): x(k+1) = Ad * x(k) + Bd * u(k), where Ad = exp(A * Ts), and if A is invertible B = inv(A) * (Ad - I) * B
